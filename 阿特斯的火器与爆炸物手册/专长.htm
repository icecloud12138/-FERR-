火器运用

先决条件：无

你学会了使用火器的方法，获得以下增益：

l 你的敏捷加1，上限为20。

l 你获得火器熟练。

l 当你使用火器命中检定骰出1时，可以重新骰值。你必须接受第二次骰值的结果。你只能在进行一次短休或一次长休后再次发动此增益。

火器大师

先决条件：火器熟练

你对火器的运用了如指掌，获得以下增益

l 你使用拥有熟练项的火器时，忽略其装填属性。

l 你执行攻击动作以单手武器攻击后，可以用附赠动作使用一把正持握的短火器发动攻击。

l 你可以使用一个附赠动作消除武器的哑火状态。

枪手技巧

先决条件：火器熟练，人物等级2级

你掌握枪手们运用火器的方法，你习得3项技巧能力，每升4级时你获得一个额外技巧能力，获得技巧能力时你还可以替换掉一个已习得的技巧能力。你获得若干技巧点用于发动特殊的枪手能力，技巧点数上限为你的熟练加值。每当你在战斗中通过火器打出一次重击或通过火器将一个敌对生物的生命值降为0，你都可以回复一个被消耗的技巧点。你也可以在完成一次长休后重新回复所有技巧点。技巧能力豁免DC=8+你的熟练加值+你的敏捷调整值。以下是技巧能力列表。与能力对应的先决条件，是获得该能力的必要条件与充分条件。

死亡之眼。你可以消耗1技巧点使你火器的下一次远程武器攻击中获得优势。你一回合内只能使用一次此技巧能力。

闪避。你可以消耗1技巧点使用一个附赠动作执行疾走、撤离或回避动作。你只能在穿着轻甲或无甲时使用此能力。

快速上弹。你可以消耗1技巧点立即装填你的火器。

自由射击。你可以消耗1技巧点使用火器以一个附赠动作进行一次远程攻击。

枪柄打击。你可以消耗1技巧点使用一个附赠动作用枪托或枪柄进行近战攻击或执行推撞。此近战攻击视为将火器作为具有熟练项的近战武器使用。短火器造成1d6点钝击伤害，长火器造成1d8钝击伤害。

干扰射击。先决条件：人物等级第4级。你可以消耗1技巧点使用火器以一个动作对一个生物发动一次远程攻击，命中后造成正常伤害，且该生物的下一回合发动攻击时具有劣势。你一回合内只能使用一次此技巧能力。

协助射击。先决条件：人物等级第4级。你可以消耗1技巧点使用火器以一个动作对一个生物发动一次远程攻击，命中后造成正常伤害，且其他生物针对该生物发动的下一次武器攻击具有优势。你一回合内只能使用一次此技巧能力。

威吓射击。先决条件：人物等级第16级。你可以消耗2技巧点使用火器以一个动作进行一次威吓射击。半径30尺内的所有敌对生物必须进行一次感知豁免检定，失败将受到同等与法术恐惧术效果的影响。你一回合内只能使用一次此技巧能力。

穿透射击。先决条件：人物等级第16级。你可以在投攻击检定前消耗2个技巧点，如果击中目标，你可以对一个站在该目标后面的另一个距离你不大于你火器常规射程内的目标进行一次带有劣势的攻击检定，命中时其伤害减半（向下取整）。你一回合内只能使用一次此技巧能力。

眩晕射击。先决条件：人物等级第16级。当你使用火器击中目标时，你可以花费3技巧点使目标陷入震慑直到你的下回合开始。

失血重击。先决条件：人物等级第16级。你用火器打出重击时，可以消耗3点技巧点使目标在它的回合结束时受到这次攻击一半（向下取整）的黯蚀伤害，此黯蚀伤害无视目标的免疫或抗性，此伤害对不死和构装生物无效。

连续射击。先决条件：人物等级第16级。你可以消耗1点或至多你当前熟练加值的技巧点使用火器以一个动作对一个或多个目标发动连续攻击，其连击次数等于消耗的技巧点数，从第二次攻击开始，每次攻击承受-2减值，当你的连击出现未命中或装填数为0时，连击也会提前停止。你一回合内只能使用一次此技巧能力。

火力压制者

先决条件：枪手技巧

当你使用具有爆射、扫射属性的火器进行攻击时，生物对抗你爆射或扫射的豁免成功时，它仍将受到一半伤害。你可以消耗至多5点技巧点增加爆射或扫射的威力，每消耗1点增加1点DC和1点武器类型的伤害。

快拔枪手

先决条件：枪手技巧

你掌握快速拔枪和对决的技巧使你能够快速应对突发情况。你获得以下增益：

l 你的先攻调整值+2。每回合第一次掏出或收回短火器时不消耗动作。

l 你使用火器时在敌对生物5尺范围内发动远程攻击检定时，不会因此而具有劣势。

l 你可以使用火器对离开和进入你10尺范围的生物进行借机攻击。

速射枪手

先决条件：快拔枪手

你可以消耗2点技巧点使用短火器以一个动作对你面前15尺锥形区域内的所有敌对生物发动一次远程武器攻击，当装填数为0时提前结束攻击。你一回合内只能使用一次此特性。

快枪手直觉

先决条件：速射枪手

当你周围5尺内的生物对你发动一次近战攻击时，你可以消耗反应动作将自己的感知调整值加入至本次攻击的AC中，若该生物本次攻击失手，则作为反应的一部分，你可以立即移动等同于你移动速度的距离而不引发借机攻击。

精确射手

先决条件：枪手技巧，神射手

你的精确射击能够在命中敌人要害时造成更严重的伤害，你获得以下增益：

l 计算使用火器进行远程武器攻击的重击伤害时，你可以额外骰一个所用武器的伤害骰。

l 如果你在本回合没有进行移动，你可以消耗1点或至多4点技巧点使你变得更精确。你使用火器进行的下一次远程武器攻击将造成额外消耗的技巧点数d6的伤害。发动此特性后你的速度降为0直到下一回合开始。你一回合内只能使用一次此技巧能力。

致命枪手

先决条件：精确射手

在攻击检定前，你可以消耗1技巧点并瞄准一个生物身体上的特殊部位，如果特殊部位无法被看见或者目标缺少此部位，只处理普通伤害，没有额外效果。

l 头：击中后，生物受到普通伤害并且必须通过一次体质豁免鉴定，否则生物在你的下一回合开始前的所有命中检定都具有劣势。

l 手臂：击中后，生物受到普通伤害并且必须通过一次力量豁免鉴定，否则生物掉落一个由你决定的手持物品。

l 腿部：击中后，生物受到普通伤害，如果该目标为大型或更小体型则必须通过一次力量豁免鉴定，豁免失败则其应击倒地。

l 躯干：击中后，生物受到普通伤害，如果该目标为大型或更小体型则必须通过一次力量豁免鉴定，豁免失败则从你的方向向后推离10尺的距离。

l 翅膀：攻击命中时，生物普通受到伤害且进行一次体质豁免检定，否则生物在你的下一回合开始前飞行速度降为0。

枪匠

先决条件：无

你通过了解火器的复杂结构而学习如果制作枪支弹药和爆炸物，你获得修补工具熟练。

你可以用修补工具花费相当于其价格的一半的原材料以每50gp(向下取整)/天（最少1天）的时间制作火器。例如你要制作一把价值为350gp的半自动手枪时，你需要准备价值为175gp的原材料，随后消耗3天的时间来制作这把手枪。

制作爆炸物和弹药只需要花费其价格的一半的原材料以每5gp（向下取整）/1个小时（最少1个小时）的时间制作多份爆炸物和弹药，同时间可以制作不同的弹药和爆炸物。例如制作20颗子弹加上10颗霰弹就需要总共4gp的原材料并且消耗1个小时的时间制作；而制作一枚炸弹就需要花费75gp并且消耗15个小时的时间去制作。

枪匠发明家

先决条件：枪匠

在魔法的领域，法师是操纵和发明魔法的专家。而相同的，火器的发展少不了那些充满着奇特想法的枪匠们。你可以使用智力代替使用火器发动远程武器攻击的命中和伤害所使用的敏捷属性。你可以使用修补工具花费物品原材料的金钱和时间制造特殊的物品，这些物品在使用了你智力调整值+熟练加值（最少为1）的次数后会因损坏而被销毁。

l 钩爪枪。制作花费30gp，制作耗时2小时。你可以使用一个动作将钩爪发射到至远50尺的一个可以供钩爪固定的点，由一条最长50尺长的绳子连接着钩爪和钩爪枪。你可以使用绳子进行攀爬。你也可以使用一个动作抓取一个至多50尺内一件不重于10磅的物件并将其拉回。

l 捕网枪。制作花费10gp，制作耗时2小时。你可以使用一个动作将捕网发射至多30尺的地方。一只被捕网命中的大型或更小体型生物将被束缚，束缚状态持续至该生物解脱。捕网不能影响无形体生物，以及巨型或更大体型的生物。一个生物可以用动作进行一次DC10的力量检定，以解脱自己或其触及范围内另一生物的束缚状态。对网造成5点挥砍伤害（AC10）同样可以解脱生物的束缚状态而不伤害及生物本身，并以此摧毁捕网进而终止束缚效应。

l 发射器。制作花费50gp，制作耗时2小时。你可以使用一个动作将一个不重于2磅重的小物件发射到至多120尺远的地方。你也可以将炸弹、炸药、手榴弹或者火球项链等类似物品发射出去并激活其效果。

l 烟雾弹。制作花费40gp。制作耗时10分钟，你可以使用一个动作将烟雾弹扔到至多60尺远的一点。落地再经过一轮后，它将释放出一团烟雾，并在20尺半径范围内形成一片重度遮蔽的区域。一阵和风（风速至少每小时10里）可以用4轮时间驱散该烟雾区域，而一阵强风（风速至少每小时20里以上）则只需1轮时间将其吹散。你可以用发射器发射此物品。此物品发动效果后立即销毁。

枪支改装师

先决条件：枪匠发明家

你可以修改你自己的火器使其加强，你可以在休整期或长休期间使用修补工具花费金钱为你的火器附加以下增益,每一把火器武器最多附加你的熟练加值一半（向上取整）的次数。你无法为魔法武器附加增益。你也可以在休整期或长休期间使用修补工具将相应的增益取消，取消增益不会将此前花费的金钱返还。

l 瞄准镜。150gp。仅长火器可附加此增益，将弹药属性的最大射程变为原来的3倍。

l 快速装药。200gp。此增益无视火器的装填属性。

l 增强威力。180gp/+1，2400gp/+2（仅11级起），24000gp/+3（仅15级起）。你在使用此增益的火器发动的攻击检定和伤害掷骰获得加值，具有爆射和扫射属性武器的DC也获得加值，加值由其增益花费的金钱决定。你无法在低于其枪手职业等级要求时为武器附加相应的增益。你无法为同一把武器附加多个此增益。

l 增强可靠性。200gp。此增益无视火器的哑火属性。

l 枪托。200gp。当你第一发射击未命中目标时，本回合内下一次射击具有优势。

进阶枪匠

先决条件：枪匠发明家

当你制作弹药时可以花费更多时间和金钱制作特殊弹种，这些特殊弹药除了可以造成正常的伤害以外还有特殊的能力。

l 燃烧弹。1gp/1颗，额外造成2d6火焰伤害，未被着装或携带的可燃物件被命中时将被点燃。

l 曳光弹。2gp/1颗，命中后该生物进行一次体质豁免，否则目盲直到你的下回合开始。

l 爆鸣弹。5gp/1颗，射击目标周围5尺范围内生物进行一次体质豁免，否则受到2d10雷鸣伤害。

l 穿刺弹。5gp/10颗，无视目标的穿刺抗性。

l 凝滞弹。5gp/1颗，命中后该生物进行一次体质豁免，否则其下一回合开始速度减半。

l 破魔弹。3gp/1颗，对邪魔/不死生物额外造成2d8光耀伤害。

同一把火器在一个回合内不能连续射出相同的弹药，否则此次攻击自动哑火，免疫哑火的增益无法免疫这次攻击的哑火。例如你可以以第一发爆鸣弹，第二发燃烧弹，第三爆鸣弹的组合射出子弹，但是第一发爆鸣弹和第二发爆鸣弹的组合会在第二发射击之前哑火导致此次攻击失败。以上豁免所使用的DC=8+你的熟练加值+你的智力加值。
